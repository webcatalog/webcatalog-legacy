Known Bugs:

  • Google Chrome & Chromium
    • Linux: High-res icon is displayed in launcher. However, Chrome will override Taskbar icon with the page icon upon load. This lowers your Taskbar icon resolution #9.
    • macOS: Dock icon launches Chrome instead of the app if all windows are closed #8.
  • Firefox
    • Firefox doesn't have a special mode for standalone apps so you'll need to manually hide UI elements in your application's profile to make using your 'web app' more pleasant #7.
